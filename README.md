# Template: Responsive Layout (Design 01)

A simple and responsive layout consisting of a header, footer, & nagivation menu.

Contains a single breakpoint at 768px width.

## Dependecies

* React 18.1.0
* react-dom 18.1.0
* react-router-dom 6.3.0
* sass 1.53.0
